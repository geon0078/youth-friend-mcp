# 청년친구 MCP v3.0

청년정책, 청년센터, 고용24 정보를 통합 제공하는 MCP(Model Context Protocol) 서버입니다.

## 기능 (22개 도구)

### 온통청년 - 기본 검색 (4개)

#### `search_youth_policies`
청년정책을 검색합니다.

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| keyword | string | - | 검색 키워드 (정책명에서 검색) |
| region | string | - | 지역명 또는 지역코드 (예: 서울, 11) |
| category | string | - | 정책 대분류 (일자리, 주거, 교육, 복지문화, 참여권리) |
| subcategory | string | - | 정책 중분류 |
| pageNum | number | - | 페이지 번호 (기본값: 1) |
| pageSize | number | - | 페이지 크기 (기본값: 10, 최대: 100) |

#### `get_policy_detail`
정책번호로 상세 정보를 조회합니다.

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| policyNo | string | O | 정책번호 |

#### `search_youth_centers`
전국 청년센터를 검색합니다.

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| region | string | - | 지역명 또는 시도코드 (예: 서울, 11) |
| sggCd | string | - | 시군구코드 |
| pageNum | number | - | 페이지 번호 (기본값: 1) |
| pageSize | number | - | 페이지 크기 (기본값: 10) |

#### `get_center_detail`
센터 일련번호로 상세 정보를 조회합니다.

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| centerSn | string | O | 센터 일련번호 |

## 환경변수 설정

API 키를 환경변수로 설정해야 합니다.

### 1. 환경변수 파일 생성

```bash
cp .env.example .env
```

### 2. API 키 발급 및 입력

`.env` 파일을 열고 실제 API 키를 입력하세요. (총 7개)

| 환경변수 | 발급처 | 설명 |
|----------|--------|------|
| `YOUTH_POLICY_API_KEY` | [온통청년](https://www.youthcenter.go.kr) | 청년정책 API |
| `YOUTH_CENTER_API_KEY` | [온통청년](https://www.youthcenter.go.kr) | 청년센터 API |
| `WORK24_JOB_POSTING_API_KEY` | [고용24](https://www.work24.go.kr) | 채용정보 API |
| `WORK24_SMALL_GIANT_API_KEY` | [고용24](https://www.work24.go.kr) | 강소기업 API |
| `WORK24_JOB_INFO_API_KEY` | [고용24](https://www.work24.go.kr) | 직업정보 API |
| `WORK24_TRAINING_CARD_API_KEY` | [고용24](https://www.work24.go.kr) | 국민내일배움카드 API |
| `WORK24_EMPLOYMENT_PROGRAM_API_KEY` | [고용24](https://www.work24.go.kr) | 취업역량프로그램 API |

## API 정보

이 MCP는 다음 Open API를 사용합니다:

### 온통청년 (2개)
- 청년정책 API: 청년 지원 정책 정보
- 청년센터 API: 전국 청년센터 위치 및 연락처

### 고용24 (10개)
- 채용정보 API: 구인 정보
- 강소기업 API: 인증 강소기업 정보
- 직업정보 API: 직업별 전망, 임금, 역량
- 직무정보 API: NCS 기반 직무 설명
- 국민내일배움카드 훈련과정 API
- 사업주훈련 훈련과정 API
- 국가인적자원개발 컨소시엄 훈련과정 API
- 구직자취업역량 강화프로그램 API
- 일학습병행 훈련과정 API
- 공통코드 API: 지역코드, NCS코드

## 버전 히스토리

### v3.0.0
- 12개 API 통합 (온통청년 2개 + 고용24 10개)
- 22개 도구로 확장
- 고용24 API 연동:
  - 채용정보 검색
  - 강소기업 검색
  - 직업정보 검색
  - 훈련과정 검색
  - 취업역량 프로그램 검색
- 통합 검색 기능 추가:
  - 지역 기반 올인원 검색
  - 강소기업 + 정책 매칭
